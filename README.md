# batch
Batch files I find useful

<hr>

- pdf2tiff2txt.bat
  - Used to convert pdfs to tiff to txt files
  - uses image magick/ghostscript & tesseract
- fix_system.bat
  - run both sfc /scannow and DISM scan for fixing system files
- php_server.bat
  - start php server for development
- randpass.bat
  - generate random password via powershell
